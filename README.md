# Suiteki-app

---
## 介绍

> 小米手环密钥获取工具
> 基于[Suiteki-Lib](https://github.com/sky130/Suiteki/)开发

![](/app.jpg)

---

## 优点
- 支持在手机端一键获取密钥
- 支持多款官方应用 **ZeppLife** , **小米运动健康** , **小米穿戴**
- 支持一键获取多个手环密钥
